# Omnicare (omnicare)
Omnicare is a long-term care pharmacy services company providing pharmacy services to long-term care facilities, senior living communities, and other healthcare providers. Customer-facing portals (Omniview, MyOmniview) exist for clients but no public developer APIs have been documented at this time.
